# yuma-portfolio

> My slick Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```

## firebase deploy

```
npm install -g firebase-tools
npm run generate
firebase deploy
```
